# 欢迎使用 Markdown 文档服务

这是一个支持热更新的 Markdown 文档渲染服务。

## 主要特性

- ✅ 支持热更新，刷新页面即可看到新文档
- ✅ 代码语法高亮
- ✅ 自动生成目录导航
- ✅ 全文搜索功能
- ✅ 移动端友好的响应式设计

## 如何使用

### 添加新文档

只需将 `.md` 文件放入 `public/markdown` 目录即可。

### Front Matter 支持

在文档开头添加元数据：

```yaml
---
title: 文档标题
date: 2024-03-10
---
```
